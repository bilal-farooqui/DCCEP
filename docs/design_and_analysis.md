# Design & Analysis Document

## Project Overview
CEP DC Microservices Project.

## System Architecture
- **User Service**: Handles user registration, authentication, and profiles.
- **Product Service**: Manages product catalog, inventory, and categories.
- **Order Service**: Processes orders, payments, and tracking.
- **Frontend**: React-based user interface.

## Database Design
- MongoDB is used across all services for scalability and flexibility.
